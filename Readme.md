# WebGL Application

## Schritte zur Erstellung der Conda-Umgebung und Installation der Pakete

1. **Conda-Umgebung erstellen:**

    ```
    conda create --name WebGL
    conda activate WebGL
    conda install -c conda-forge nodejs
    npm install -g npm@10.2.3
    npm install -g http-server
    ```

2. **Projekt-Setup:**

    ```
    cd <Projektordner>
    cd public
    http-server
    ```
